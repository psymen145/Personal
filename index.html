Making a personal website with ChatGPT

7/9/2023 : 30 mins eng time

I'm starting to test out the ChatGPT code interpreter plugin. Apparently it's significantly better than just regular ChatGPT. There isn't really a clear metric to gauge if this is true but I'm going with it...

You can upload the entire project directory into your thread with ChatGPT and it'll have all of the context. This was an annoyance in the regular version. It was good at solving small problems but tasks that require more context it'll struggle with... just like any human would.

You'll still need some intuition of what is right or wrong... and just coding in general. For example, I asked it: I want to add an "article" to my "What I'm Learning" tab. Maybe just have a link to the article name, and when you click the link, it'll have the actual article. I want the article to be named "Making a personal website with ChatGPT"

It gave me a good answer but it doesn't spit out the updated file. You'd need to know where to put the script tag, where to put the code snippets, etc. It's trivial but it might be confusing for someone just starting out. You can keep asking it where to put the code snippet but it'll take longer to get an MVP out.

So what I did today solely with ChatGPT prompts:
  • Ability to add articles
  • Make the site responsive. I gave it this prompt: I want to make it responsive. How can I fix it, I think I'm not using container correctly in material css
    It gave me the correct response but made the content alittle too narrow on desktop. That was an easy fix
  • Asked it to help me collapse my header into a hamburger on small screens. It originally suggested to use material css' navbar. The asthetic is completely different from what I wanted. This took a majority of my time. I actually have no idea how to create these responsive hamburgers and cross icons from scratch. So I spent probably 10 mins of back and forth with ChatGPT. It would regress several times. 2 Steps forward 1 step back. Sometimes it would leave our lines of code that were perfectly fine in the previous response.
So what I did today without prompts:
  • Hacking together some fixes for material css. I couldn't get the bulletpoints showing because of material css. I just overrode it with important
  • Fixing the spacing after asking it to make my site responsive. It added offset on medium and large screens for some reason. I just removed those classes
